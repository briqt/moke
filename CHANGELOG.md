# Changelog

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.9]

- **终端**：「文本段」输入改为在底部就地内联展开（替代原来的独立弹层）——打开 / 关闭更顺滑，软键盘不再收起后重新弹出。

## [0.1.8]

- **连接页**：固定按「项目」分组，去掉标题栏的分组 / 排序选择。分组可折叠 / 展开且记住折叠状态（重启保留）；长按分组头可拖动调整分组顺序；组内主机可长按拖动排序（未分组作为普通分组）。
- **会话页**：分组维度调整为「无 / 按连接 / 按项目」，排序维度调整为「创建时间 / 更新时间 / 手动」（均为组内排序）；分组之间可长按拖动排序、分组可折叠。
- **tmux 会话管理**：终端详情页新增 tmux 入口——连接后自动检测远端 tmux 会话，检测到则在顶栏显示带会话数的入口（检测不到不显示）；可查看会话列表并新建 / 重命名 / 关闭 / 附加（当前仅 SSH）。
- **终端**：进一步修复「登录后自动执行命令」偶发首字符丢失 / 回显重复——改为等终端输出稳定后再发送、并以回车符提交。

## [0.1.7]

- **会话标题**：改为双行——第 1 行动态标题、第 2 行连接名（设备名）· 协议徽标 · 状态；去掉与设备名重复的 `user@host`。移除「标题前缀」功能（分组已替代其作用）；「修改标题」仍可用（长按会话卡 / 终端 ⋮ 菜单）。
- **分组 / 排序**：连接页与会话页统一为「分组 + 排序」两个独立维度，收进标题栏右上角两个下拉——分组（连接页：无 / 项目；会话页：无 / 项目 / 主机 / 日期）、排序（名称 / 最近 / 手动）；按项目分组时分组标题已展示、卡片副标题不再重复分组名。
- **连接编辑**：「分组」改为可编辑下拉——点开选择已有分组，或直接输入新建；无主机使用的分组自动消失。「登录后自动执行」支持多行（每行一条命令）。
- **终端**：修复「登录后自动执行命令」偶发首字符丢失 / 命令错乱（改为待 shell 就绪后再发送）；选中文字菜单中文化并移除无实际作用的「更多」项；附加键回车改为文字 `Enter`、「文本段」入口改为图标；顶栏高度与其它页一致；从终端返回列表时自动收起软键盘。

## [0.1.6]

- **会话标题**：会话默认标题前缀采用创建时所用连接的名称（连接未命名则无前缀），多会话更易区分；动态标题叠加在其后。
- **会话列表**：会话卡新增「复制」——一键以同一连接再开一个独立会话，沿用原标题 / 前缀并附加不重复编号（如 `(2)`）以便区分。

## [0.1.5]

- **会话标题**：新增自定义标题与标题前缀——会话列表长按、终端 ⋮ 菜单均可设置；自定义标题优先级最高（完全覆盖），前缀叠加在动态标题之前。mosh 会话自动去除原生 `[mosh]` 前缀（协议已由徽标标识）。
- **会话列表**：卡片改为与终端顶栏同款双行布局（标题 + `user@host` · 协议徽标 · 延迟）。
- **终端**：右上角 ⋮ 菜单新增「弹出键盘 / 关闭连接」并收紧字号；粘滞修饰键 Ctrl / Alt 改为一次性（用一次即自动取消）；协议徽标进入顶栏（mosh 强调色高亮）；「发送并回车」改为文本与回车分两次发送，全屏 TUI（raw 模式）下也能正确提交而非只插入换行；底部附加键更扁、方向 / 回车符号放大。
- **界面**：各页顶栏高度收窄；底部导航去除选中态背景块、图标与文字间距收紧；外观页字号 / 间距 / 滑块更紧凑；协议徽标进一步压小、贴合文字；语言选择弹窗更紧凑；关于页收紧并移除开源许可区块；默认字号 11 → 11.5。

## [0.1.4]

- **多语言**：新增中英双语（i18n），默认跟随系统语言，设置页内可切换（跟随系统 / 中文 / English），即时生效。
- **界面**：设置各层级页标题字号统一协调；底栏图标 / 文字整体下移微调；外观实时预览示例改为更贴近真实终端的会话（提示符 + 彩色 `ls` + 代码行 + 调色板）。
- **表单**：新建 / 编辑连接的输入更紧凑精致；终端「文本段」输入框收敛留白并去掉冗余提示。
- **字体管理**：移除页内大预览（预览已在外观页），列表更聚焦。
- **文档**：README / CONTRIBUTING 提供中英双语（`*.md` 英文 · `*.zh-CN.md` 中文）。

## [0.1.3]

- **排版滑块**：行距 / 字间距范围改为绕 1.0 对称（0.7–1.3），默认值落在滑轨正中；字号上限收窄至 24（8–24），滑轨都落在实际可用区间。捏合缩放上限同步为 24。

## [0.1.2]

- **外观**：设置页按「字体 / 排版 / 配色 / 光标」分组；字号 / 行距 / 字间距改为滑块 + 步进微调；新增「恢复默认」（可撤销）；实时预览修复色带丢失并按内容收敛高度。
- **字号**：默认 11；全局 0.5 步进（设置、终端菜单、捏合缩放一致）。
- **字体管理**：已下载 / 已上传的字体排在列表前面；上传本地字体时显示进度与结果提示。
- **连接 / 会话**：支持长按拖动排序（连接页新增「手动」排序方式，顺序持久化）。
- **终端**：右上角新增折叠菜单（字号增减 / 恢复默认字号 / 显隐底部快捷键）。
- **配色**：精简为暗色方案。
- **关于**：页面信息精简。

## [0.1.1]

- **字体**：内置符号回退字体，补主/回退字体常缺的媒体 / 几何 / 杂项符号等终端字形；可下载字体扩充至十余款主流开源等宽（IBM Plex Mono、Source Code Pro、Cascadia Code、JetBrainsMono Nerd Font 等）；字体管理支持上传本地 TTF / OTF。
- **终端**：外观设置对运行中的会话即时生效；顶栏改为双行（会话名 + `用户@主机 · 协议 · 延迟`），连接信息不再单独占用终端区域；底部附加键更扁平，键位调整（右端为「文本」段入口与回车）。
- **界面**：二级页顶栏高度统一瘦身；字体预览区按内容高度收敛；文本段输入框限高、发送按钮常驻。
- **图标**：更新为水墨风格「M」应用图标。

## [0.1.0]

- **SSH**：密码 / 私钥认证；主机密钥 TOFU 校验；跳板机（ProxyJump）；登录后自动执行命令；窗口 resize；保活心跳。
- **mosh**：随包 native `mosh-client`，独立子进程 + PTY 运行，UDP 漫游。
- **多会话**：会话常驻、跨页切换；前台服务在后台保持连接。
- **连接管理**：分组 / 排序；协议徽标；复制连接命令 / 创建副本。
- **终端**：双排附加键 + 文本段输入；复制粘贴；捏合缩放；顶部状态条（协议 / 主机 / 延迟）。
- **外观**：实时预览；多套配色；字体管理（内置思源黑体子集作中文回退，可下载 Fira Code / Maple / Hack）；行距 / 字间距可调。
- **安全**：连接凭据经 Android Keystore（AES-GCM）加密存储。
